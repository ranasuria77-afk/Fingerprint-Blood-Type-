From print to Type
what if a single fingerprint could reveal something as critical as a person’s blood
group?
Print2Type explores an innovative application of Deep Learning by analyzing fingerprint
images and transforming them into meaningful predictions through AI-driven image
analysis. Instead of treating fingerprints as simple biometric identifiers, the project
investigates whether complex fingerprint patterns can be learned and classified into the
eight blood group categories using a CNN-based architecture.
Through preprocessing, pattern extraction, and intelligent classification, the system
demonstrates how artificial intelligence can uncover valuable information from biometric
data and convert it into an interactive prediction system. Print2Type combines computer
vision, deep learning, and real-time deployment into a complete end-to-end AI solution
